# 🐱 말랑콩냥 키우기 (WinForms 클리커 게임)

C# Windows Forms를 활용해 제작한 간단한 방치형 클리커 게임입니다.  
고양이 캐릭터를 클릭해 재화를 획득하고,  
사료와 뽑기 시스템을 통해 수익을 증가시키는 구조의 게임입니다.

---

## 📌 프로젝트 개요

- 프로젝트명: 말랑콩냥 키우기
- 장르: 방치형 / 클리커 게임
- 개발 환경
  - Language: C#
  - Framework: .NET (Windows Forms)
- 플랫폼: Windows PC
- 개발 목적
  - WinForms 기반 UI 구성 및 이벤트 처리 이해
  - 간단한 게임 로직(확률, 수익 계산) 구현 경험

---

## 🎮 게임 설명

메인 캐릭터인 콩냥이를 클릭하면 돈을 획득할 수 있으며,  
초당 수익 시스템을 통해 방치 상태에서도 재화가 증가합니다.

상점에서 사료와 뽑기를 구매할 수 있고,  
뽑기를 통해 획득한 고양이를 메인 캐릭터로 설정할 수 있습니다.

---

## 🖥 화면 구성

- 메인 화면
  - 콩냥이 클릭
  - 현재 돈 및 수익 확인

- 상점 화면
  - 사료 구매
  - 고양이 뽑기

- 도감 화면
  - 보유 고양이 목록 확인
  - 고양이 클릭 시 메인 캐릭터로 설정

> 화면 전환은 Panel을 이용해 구현

---

## 🐾 캐릭터 시스템

### 등급별 고양이

#### A 등급
- 기본냥이, 호냥이, 회냥이
- 기본 수익: 90원
- 강화 수익: +15원

#### S 등급
- 비니냥이, 폭죽냥이, 중절냥이
- 기본 수익: 180원
- 강화 수익: +45원

#### SR 등급
- 여왕냥이, 왕냥이, 할리퀸냥이
- 기본 수익: 450원
- 강화 수익: +150원

---

## 🍖 사료 시스템

사료는 일정 시간 동안 수익을 증가시키는 아이템입니다.

- 일반 사료  
  - 5초간 수익 1.35배 증가

- 중급 사료  
  - 5초간 수익 1.75배 증가

- 고급 사료  
  - 5초간 수익 3배 증가

### 사료 사용 규칙

- 사료 효과는 중첩 불가
- 효과 종료 시 알림 창 출력
- 남은 사료 개수 알림 제공

---

## 🎲 뽑기 시스템

돈을 사용해 확률에 따라 고양이를 획득할 수 있습니다.

### 뽑기 확률

- 일반 뽑기
  - SR: 15%
  - S: 35%
  - A: 50%

- 중급 뽑기
  - SR: 20%
  - S: 50%
  - A: 30%

- 고급 뽑기
  - SR: 50%
  - S: 35%
  - A: 15%

### 뽑기 규칙

- 소지 금액이 부족할 경우 알림 창 출력
- 구매 시 확률에 따라 고양이 랜덤 지급
